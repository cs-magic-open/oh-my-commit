# 即将推出

### 1. 团队协作功能 🔜
预计发布时间：2024 Q2

  - 团队级配置管理

- **协作工具集成**
  - GitHub/GitLab 集成

预计发布时间：2024 Q3

- **自定义模型支持**
  - 私有模型部署

  - 深度依赖分析

预计发布时间：2024 Q4

- **实时协作视图**
  - 团队活动面板
